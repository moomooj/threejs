# Vite 베이스 Three.js 기본 셋업 입니다.


``` bash
# git clone
git clone https://github.com/moomooj/threejs.git

# dependencies install
npm install

# 로컬 서버에 run 하기
npm run dev

# dist 폴더에 빌드하기
npm run build
```

## src/script 파일 내 three.js 코드 작성하시면 됩니다.

``` javascript I'm A tab
import * as THREE from "three";

```
